# Examples

Example applications built with PromptingNemo.

> **Note:** Each example has its own dependencies beyond the core `promptingnemo` package. See the individual README and `requirements.txt` in each directory before running.

| Example | Description |
|---------|-------------|
| [voicebot](voicebot/) | Voice-enabled chatbot with ASR, TTS, emotion detection, and multi-language support |
| [iot_assistant](iot_assistant/) | AI-powered virtual assistant for IoT tasks -- email, calendar, Spotify, WhatsApp, weather, and more |
| [wellness_companion](wellness_companion/) | Wellness companion chatbot that provides personalized recommendations using semantic search and LLMs |
